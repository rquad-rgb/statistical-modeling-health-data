# Statistical Modeling of Health Data

## Data
The dataset used in this project was provided for academic purposes and is not publicly available.  
The analysis is included to demonstrate statistical modeling techniques.

## Overview
This project analyzes relationships between body composition variables using statistical modeling techniques in R.

## Objectives
- Explore relationships between variables
- Build regression models
- Perform ANOVA and hypothesis testing
- Interpret results in a real-world context

## Methods Used
- Linear Regression
- Multiple Regression
- ANOVA
- Two-Way ANOVA (Interaction Analysis)

## Key Findings
- Body protein content has a strong negative relationship with body fat ratio  
- Height and weight significantly predict total body water  
- Age groups show differences in body fat distribution  
- Diabetes and hepatic fat both influence triglyceride levels, with no interaction effect  

## Tools & Libraries

- dplyr
- car
- lmtest
